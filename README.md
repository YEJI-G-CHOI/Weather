# Expo Weather App

Expo 를 사용해 만든 날씨앱입니다.

3시간 단위로 5일 동안의 날씨를 확인할 수 있습니다.

Expoを使用して作った天気アプリケーションです。

一日３時間ごとに、５日間の天気が確認できます。
